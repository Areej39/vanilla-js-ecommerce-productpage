# Vanilla JS E-Commerce Product Page

A responsive e-commerce product page built using HTML, CSS, and JavaScript. The application dynamically displays products from a JavaScript array, allowing users to browse products and add them to a cart with a live cart counter update.

## Features

- Responsive design for desktop, tablet, and mobile devices
- Dynamic product rendering using JavaScript
- Product data stored in an array of objects
- Product cards generated with JavaScript
- Add to Cart functionality
- Live cart item counter
- Clean and modern user interface
- Easy to extend with additional products

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript

## How It Works

1. Product information is stored in a JavaScript array of objects.
2. JavaScript dynamically generates product cards and displays them on the page.
3. Users can add products to the cart by clicking the "Add to Cart" button.
4. The cart counter updates automatically whenever a product is added.
5. The layout adapts to different screen sizes using CSS media queries.
